    Esse foi um projeto realizado apenas para pratica de programação.

    Como funciona e qual o intuito?

    O projeto foi feito para automatizar a criação de papeletas de licença, onde era feito manualmente.
    O sistema permite selecionar nomes de militares e horários em um layout interativo. A partir dessas seleções, ele gera automaticamente papeletas personalizadas com base em um modelo padrão, preenchendo os dados dos militares, armazenados na memória do programa, juntamente com a data e hora atuais. Todo o processo é feito de forma automatizada, poupando tempo e reduzindo erros manuais.
